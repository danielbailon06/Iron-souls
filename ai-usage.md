## AI prompts related to JavaScript development

AI was also used to assist with coding and game logic during development. 
Below are examples of prompts used during development.

### 1. Scene transitions
> How can I change from one scene to another in a JavaScript game when the player enters a specific area of the map?

### 2. Interaction system
> How can I display a message when the player approaches a certain area and allow interaction using the "E" key?

### 3. Enemy AI
> How can I implement a simple enemy AI that follows the player and attacks when in range?

### 4. Boss mechanics
> How can I implement a boss enemy in JavaScript that moves randomly and shoots projectiles toward the player?

## AI assistance in development

AI was also used to help with:
- Brainstorming the game concept and visual style
- Helping write and improve the text for the README and presentation
- Translating some content
- Suggesting boss names and lore ideas
- Helping implement gameplay systems in JavaScript, such as:
  - scene transitions
  - collision hitboxes
  - enemy behavior
  - boss logic
  - HUD elements

## Notes

All AI-generated content and suggestions were reviewed, edited, and integrated manually into the final project.
