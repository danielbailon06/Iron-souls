# Iron Souls

## Description

Iron Souls is a small top-down fantasy action game built with HTML, CSS and JavaScript.

You play as a warrior exploring magical ruins, fighting enemies and progressing through different scenes until reaching the final boss: Selunara, Goddess of the Moon.

The game includes basic combat, enemy AI, scene transitions and a final boss fight.

## Instructions to play

Controls:

W A S D or Arrow Keys → Move

C → Attack

E → Interact with the environment (enter portals, triggers, etc.)

## Goal:

Move through the different areas of the map

Defeat enemies

Progress through the scenes

Defeat the final boss
